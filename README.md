# mytab
mytab.mooo.com – আপনার মোবাইল ব্রাউজারের স্মার্ট হোমপেজ। এটিি সহজ, দ্রুত এবং মোবাইল-বান্ধব হোমপেজ, যা আপনার দৈনন্দিন ইন্টারনেট ব্যবহারের অভিজ্ঞতাকে আরও আরামদায়ক করে তোলে। এখানে আপনি পাবেন পত্রিকা, রেডিও, টিভি, সামাজিক ওয়েব সহ প্রয়োজনীয় ও জনপ্রিয় বিভিন্ন ওয়েবসাইটের গুরুত্বপূর্ণ লিংকের সমাহার—সব এক জায়গায়, এক ট্যাপে।

মোবাইলে Chrome-এ mytab.mooo.com হোমপেজ সেট করার নিয়ম

🔹 ধাপ ১: মোবাইলে Chrome ব্রাউজার চালু করুন।

🔹 ধাপ ২: উপরে ডান পাশে থাকা ⋮ (তিন ডট) এ ক্লিক করুন → Settings (সেটিংস) এ যান।

🔹 ধাপ ৩: Settings-এর ভিতরে Homepage নামে একটি অপশন পাবেন → সেটাতে ক্লিক করুন।

🔹 ধাপ ৪: Homepage অপশনটি ON করুন।

🔹 ধাপ ৫: “Enter custom web address” বা “Open this page” অপশনে লিখুন:  https://mytab.mooo.com

🔹 ধাপ ৬: Save/Back করুন  এখন সেটিংস থেকে বের হয়ে আসুন।


আপনার প্রয়োজনীয় কোন লিংক যুক্ত করতে চাইলে উপরের +লিংক বাটনে চাপ দিন লিংকের নাম দিন যেমন "ইউটিউব" OK বাটনে চাপ দিন। এর পরে URL বক্সে লিংক টি টাইপ করুন  যেমন " https://www.youtube.com/" OK বাটনে চাপ দিন। নিচে দেখুন লিংক যুক্ত হয়েছে।
